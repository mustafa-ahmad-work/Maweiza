## English Version

### 📖 About The Project
**Maweiza** is a comprehensive Islamic platform designed to spread knowledge and science. We provide a diverse range of content to meet your spiritual needs, including a rich library of books, articles, accurate prayer times, and a collection of Hadiths. Our goal is to make Islamic knowledge accessible, beautiful, and user-friendly.

### ✨ Key Features
*   📚 **Rich Islamic Library**: Access a wide variety of books and articles.
*   🕌 **Prayer Times**: Stay updated with accurate prayer times based on your location.
*   ☁️ **Hadiths**: Read and explore authentic Hadiths.
*   🌗 **Dark/Light Mode**: Toggle between themes for comfortable reading day and night.
*   🖼️ **Fullscreen Mode**: Distraction-free reading experience.
*   📱 **Cross Platform**: Responsive design that works seamlessly on Desktop, Tablet, and Mobile.

### 🏁 Getting Started
To get a local copy up and running, follow these simple steps.

#### Prerequisites
Make sure you have Node.js installed.
```bash
npm install npm@latest -g
```

#### Installation
1. **Clone the repo**
   ```bash
   git clone git@github.com:mustafa-ahmad-work/Maweiza.git
   ```
2. **Navigate to the project directory**
   ```bash
   cd Maweiza
   ```
3. **Install NPM packages**
   ```bash
   npm install
   ```
4. **Start the development server**
   ```bash
   npm run dev
   ```
